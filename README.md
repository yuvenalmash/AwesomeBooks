# 📖 Awesome Books App <a name="about-project"></a>

**Awesome Books App** is a simple website that displays a list of books and allows the user to add and remove books from that list.

## 🛠 Built With <a name="built-with"></a>

### Tech Stack <a name="tech-stack"></a>

- HTML
- CSS
- JavaScript

<!-- Features -->

### Key Features <a name="key-features"></a>

- **UI is made of medium-fidelity wireframe**
- **Persistent data storage in local storage**
- **Luxon module is used for date and time**

- Paste the path to your browser
